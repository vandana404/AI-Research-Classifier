# AI Classifier model

An AI-powered Research Paper Discovery and Analysis Platform that enables semantic search, automated summarization, keyword extraction, and topic classification for machine learning research papers.

## Project Overview

AI Research Power Station helps researchers, students, and AI practitioners quickly discover relevant research papers using natural language queries.
Instead of traditional keyword matching, the system uses vector embeddings and semantic search to retrieve the most relevant papers.

## Features

### Semantic Search
- Natural language search across research papers
- Embedding generation using Sentence Transformers
- Fast retrieval using FAISS vector search

### Research Paper Summarization
- Automatic paper summary generation
- Powered by Transformer-based NLP models

### Keyword Extraction
- Extracts important concepts from papers
- Uses KeyBERT for keyword generation

### Topic Classification
- Classifies papers into AI domains
- Zero-shot classification using Transformers

### Research Insights
- Find related papers
- Explore emerging research trends
- Discover relevant topics quickly
